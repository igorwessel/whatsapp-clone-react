## Como iniciar o projeto

- Apenas clone o projeto.
- Instale as dependências com o comando abaixo no terminal dentro da pasta do repositorio.
 ```
 npm install
 ```
- Inicie o projeto com o comando abaixo no terminal.
```
npm start
```
